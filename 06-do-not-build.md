## Règle de décision

Une page de ville ne se justifie qu'à partir de **~100 recherches/mois** sur le combo service+ville.
Aujourd'hui live : **Paris** (FR), **Bruxelles/Genève** (hubs francophones). Candidats suivants : **Lyon**, **Bordeaux** (P4). Pour l'EN, la géo est un test (`/seo-agency-london`), pas une priorité.
